# Testes (tests/)

Diretorio para scripts de verificacao e testes automatizados.

## Executando Testes
A maioria dos testes aqui sao scripts manuais ou de verificacao de integracao simples.

Exemplo:
```powershell
python -m tests.verify_system
```

*(Nota: A suite de testes ainda esta em desenvolvimento e muitos testes sao scripts ad-hoc criados durante o desenvolvimento).*

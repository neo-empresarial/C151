# Features

Funcionalidades de domínio específicas.

- `inferencia/`: Motor de processamento facial. Contém a lógica que interage com a biblioteca DeepFace para detecção e reconhecimento.
- `auth/`: (Legado/Em desuso) Lógica de autenticação antiga, substituída principalmente pelo `database.py` e `state.py` na arquitetura atual.
- `cadastro/`: (Legado/Em desuso) Lógica de cadastro antiga, migrada para `src/pages/dashboard.py`.

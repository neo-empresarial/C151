# Páginas (Pages)

Módulos responsáveis pela interface do usuário (UI) usando NiceGUI.

- `login.py`: Tela principal de reconhecimento facial. Realiza a inferência em tempo real e libera acesso.
- `dashboard.py`: Painel administrativo protegido. Permite CRUD de usuários e gestão de biometria.
- `setup.py`: Tela de configuração inicial (First Run) para criação do primeiro administrador.
- `landing.py`: (Opcional/Legado) Ponto de entrada alternativo.
